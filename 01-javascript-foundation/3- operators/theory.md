# Operators

Operators perform actions on values.

Common operator groups:

- Arithmetic: `+`, `-`, `*`, `/`, `%`
- Assignment: `=`, `+=`, `-=`
- Comparison: `===`, `!==`, `>`, `<`, `>=`, `<=`
- Logical: `&&`, `||`, `!`
- Ternary: `condition ? valueA : valueB`

Prefer strict equality with `===` and `!==`.
