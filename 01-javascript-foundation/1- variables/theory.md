# Variables

Variables store values so a program can reuse and update data.

JavaScript has three common declaration keywords:

- `let` creates a block-scoped variable that can be reassigned.
- `const` creates a block-scoped variable that cannot be reassigned.
- `var` is function-scoped and is mostly avoided in modern code.

Use `const` by default, and switch to `let` only when the value must change.
