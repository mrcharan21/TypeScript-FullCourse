# Arrays

Arrays store multiple values of the same type.

```ts
const marks: number[] = [80, 90, 75];
const names: Array<string> = ["Asha", "Ravi"];
```

Common array operations include:

- `push` to add an item.
- `map` to transform items.
- `filter` to keep matching items.
- `reduce` to combine items into one result.

Typed arrays prevent adding the wrong kind of value.
