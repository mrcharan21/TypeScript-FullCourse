# Tuples

Tuples are arrays with a fixed order and known types at each position.

```ts
const user: [number, string] = [1, "Asha"];
```

Use tuples when position has meaning, such as:

- Coordinate pairs.
- API status pairs.
- Key-value pairs.
- Function return values with multiple parts.

Tuples are stricter than normal arrays because each index has its own type.
